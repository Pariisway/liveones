# WhisperChat - Voice & Video Chat Platform

WhisperChat is a modern web application that connects users for meaningful conversations through voice and video calls. Users can connect with "Whispers" (content creators, influencers, or conversationalists) for 5-minute paid calls.

## Features

### For Callers
- Browse available Whispers with profiles, ratings, and categories
- Purchase coins to start calls
- 5-minute voice/video calls with Whispers
- Rate and review calls
- User dashboard with call history and coin balance

### For Whispers
- Earn $12 per 5-minute call
- Flexible availability scheduling
- Profile customization with categories and bio
- Earnings dashboard with payment history
- Performance analytics and ratings

## Technical Stack

### Frontend
- **HTML5**: Semantic markup and page structure
- **CSS3**: Custom properties, Grid, Flexbox, animations
- **JavaScript (ES6+)**: Dynamic functionality and Firebase integration
- **Font Awesome**: Icon library
- **Google Fonts**: Typography

### Backend & Database
- **Firebase Authentication**: User authentication (email/password, Google)
- **Firebase Firestore**: NoSQL database for users, calls, ratings
- **Firebase Storage**: File storage for user avatars
- **Firebase Hosting**: Static file hosting and deployment

### Payment Processing
- **Stripe Integration**: For processing coin purchases
- **In-app currency**: Whisper Coins system

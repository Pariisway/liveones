# House of Whispers - Social Video Platform

A modern social platform for watching viral videos together with live voice chat and dating features.

## Features

- 🎥 Synchronized video watching
- 💬 Real-time voice chat (Agora RTC)
- 💘 "Shoot Your Shot" dating system
- 📱 Responsive design
- 🎨 Cyberpunk aesthetic
- 🔐 User authentication
- 📊 Google Analytics integration
- 💰 AdSense monetization
